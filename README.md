# Human Resources Management System (HRMS)

🚀 Developed for the **Odoo x Adamaas Hackathon**

An automated, user-friendly, and comprehensive Human Resources Management System designed to streamline organizational workflows, optimize employee management, and leverage data-driven insights for HR administrators and employees alike.

---

## 📌 Project Overview

Managing workforce data efficiently is critical for modern enterprises. This project introduces a robust HRMS solution built during the Odoo x Adamaas Hackathon. It bridges the gap between complex HR operational workflows and a seamless user experience, handling everything from onboarding and attendance tracking to performance evaluation.

![HRMS App Demo](https://raw.githubusercontent.com/robycode06-cmd/oddo/main/assets/demo.gif)

## ✨ Key Features

- **Employee Directory & Onboarding:** Centralized profile management with seamless document tracking and role-based access control.
- **Attendance & Time Tracking:** Automated check-in/check-out systems with detailed timesheet logs.
- **Leave & Absence Management:** Intuitive dashboard for requesting, reviewing, and approving leave applications.
- **Performance Evaluation:** Module to set goals, track progress, and submit peer or manager appraisals.
- **Analytics Dashboard:** Graphical breakdowns of key HR metrics such as employee retention, department distributions, and monthly attendance statistics.

![System Architecture / Dashboard Overview](https://raw.githubusercontent.com/robycode06-cmd/oddo/main/assets/dashboard.png)

## 🛠️ Tech Stack

- **Backend / Platform:** Odoo Framework (Python)
- **Database:** PostgreSQL
- **Frontend:** QWeb templates, XML, JavaScript (OWL Framework)
- **Deployment & Tools:** Git, GitHub
